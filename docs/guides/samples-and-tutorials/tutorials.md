# Tutorials

Khoá học này sẽ dạy bạn cách sử dụng ngôn ngữ Dart, các công cụ và APIs để xây dựng các ứng dụng. Nếu bạn muốn trải nghiệm thêm về thực hành code bạn hãy thử ở [codelabs]()

## Cơ bản
The following tours giả định một sự quen thuộc cơ bản với ngôn ngữ Dart cái mà bạn có thể nhận được từ việc xem qua [Học ngôn ngữ qua ví dụ](https://youngtailors.github.io/duck-lang/guides/samples-and-tutorials/language-samples.html) hoặc [language tour](). Tiếp theo, tìm hiểu về futures qua [asynchronous programming codelab.]()

Một khi bạn đã quen thuộc với ngôn ngữ và futures, tìm hiểu về *stream* và các *packages*, đó là nền tảng cơ bản cho hầu hết các chương trình Dart.

>[Asynchronous programming: streams](https://dart.dev/tutorials/language/streams)<br/> Sử dụng streams để quản lý chuỗi dữ liệu.

>[Install shared packages](https://dart.dev/tutorials/libraries/shared-pkgs)<br/> Tổ chức và chia sẻ code lên [Pubsite](https://pub.dev/).

## Server-side Dart tutorials
Các tutorials sau đây cho thấy cách phát triển tập lệnh(develop scripts), command-line apps và ứng dụng server có thể chạy trong máy ảo độc lập Dart VM.

>[Khởi đầu](https://dart.dev/tutorials/server/get-started)<br/> Get dart. Tạo ứng dụng standalone đơn giản, chạy trên Dart VM và AOT-compile nó thành mã máy gốc.

>[Viết command-line apps](https://dart.dev/tutorials/server/cmdline)<br/> Tìm hiểu về các tính năng mà các ứng dụng command-line cần.

>[Viết HTTP clients và servers](https://dart.dev/tutorials/server/httpserver)<br/> Sử dụng dart:io APIs để giao tiếp qua internet.

>[Connect Dart & HTML](https://dart.dev/tutorials/web/low-level-html/connect-dart-html)<br/> Bao gồm Dart script trong HTML page.

>[Thêm elements vào DOM](https://dart.dev/tutorials/web/low-level-html/add-elements)<br/> Thêm elements từ trang web và di chuyển chúng.

>[Xóa DOM elements](https://dart.dev/tutorials/web/low-level-html/remove-elements)<br/> Xóa elements từ trang web .
